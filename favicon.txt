<p class="title">The Coelacanth Quartet</p>
<p>Eons ago, I decided to write a novel. I then decided, as a linguist, I should create a number of constructed languages, and now I've decided to put this on the web. So I'm two levels away from <em>actually</em> writing the thing. One day&hellip;.</p>
<p>Although there is only one story, it appears here in five versions. On each page, there is the option to jump between these.</p>
<ul>
<li><a href="english/index.html">English</a></li>
<p>The original <em>Coelacanth Quartet</em>.</p>
<li><a href="tinellbian/index.html">Tinellbian</a></li>
<p>Each chapter of the original text, translated into a different Tinellbian language, using the native scripts.</p>
<li><a href="transliteration/index.html">Transliteration</a></li>
<p>The same as above, but using Roman letters.</p>
<li><a href="interlinear/index.html">Interlinear</a></li>
<p>The meaning of each morpheme, using the glossing rules and abbreviations <a href="../grammar/appendices/glossing/index.html">here</a>.
<li><a href="gloss/index.html">Gloss</a></li>
<p>Just the meaning of each morpheme, lacking the native pronunciations. Boring, really, but might work as freeform poetry.</p>
</ul>
