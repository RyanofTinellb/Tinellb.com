<p>There are 5 versions here, but only one story. Originally written in <em>English</em>, each chapter of the <em>Coelacanth Quartet</em> is being translated into a different Tinellbian language. The <strong>Tinellbian</strong> section uses the native scripts, while the <strong>Transliteration</strong> section uses Roman type. The <strong>Interlinear</strong> shows the meaning of each morpheme, using the glossing rules and abbreviations found <a href=../grammar/appendices/glossing/index.html>here.</a> The <strong>Gloss</strong> shows just the interlinear meanings, without the native pronunciations.</p>
