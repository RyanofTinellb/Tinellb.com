According to the rules of Sūdoku, each digit may only appear once in each row, column, and 3×3 box. This Sūdoku solver will automatically remove duplicates. The solver also allows for the highlighting of specific digits to allow for easy recognition of other patterns. This means that just about any 9x9 original sūdoku can be solved in less than ten minutes.

Setup Mode

Turn on Setup Mode to overwrite a cell completely. Turn off Setup Mode to remove specific numbers from a cell.

Use the mouse to select cells, or use the arrow keys on your keyboard to move the selection point.

Numbers can be added / removed by using the lower set of number buttons with the mouse, or by entering numbers on your keyboard.

Numbers can be highlighted or un-highlighted by use of the upper set of number buttons.
